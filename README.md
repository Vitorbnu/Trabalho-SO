#  Gerenciador de tarefas

Um monitor de sistema completo com interface gráfica para Windows, Linux e macOS.

##  Pré-requisitos

- Python 3.7 ou superior
- Sistema operacional: Windows 7+/Linux/macOS 10.9+

##  Instalação
 
 ## 1. Clone o repositório:
 
 git clone https://github.com/Vitorbnu/Trabalho-SO.git
 cd Trabalho-SO
 
## 2. Crie o ambiente virtual na pasta do projeto
python -m venv venv

## 3. Instale as dependências:

 pip install -r requirements.txt

## Execução

 python app.py
